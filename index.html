UX & UI Designer

I am an entrepreneur and UX professional. Having launched and run a successful start up, I understand the importance of crafting a good user experience and why telling a good story is key to the success of any business. I follow a rigorous yet adaptable UX process and have learnt from some of the best in the business.

User Experience

For any project I start by trying to understand a businesses core premise, goals and strategy. Using a range of tools such as remote user research software, workshops and clearly presented analysis, my aim to help build outcomes with long term value for both individuals and organisations.

Developer

My focus with code is the design, behaviour and feel of a product. Knowing HTML, CSS and Javascript has been very helpful when clients have tight budgets. But understanding the challenges developers face has been invaluable, especially for a complex project delivery involving large multifaceted teams.

Fun

In addition to getting excited about how we experience and use products, I trained as an artist, so find it very natural to develop the design and feel of a brand. I enjoy building long form, interactive stories and have done some just for fun like this. I live in Berlin, but work all over and love to paint, run and read in my spare time.
